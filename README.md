# Bovenmaat Fitness

Securely upload and share files.

## Installation

For development

1. `git clone git@github.com:Luca-Castelnuovo/BovenmaatFitness.git`
2. `yarn install`
3. Start development server `yarn start`

For production

1. `git clone git@github.com:Luca-Castelnuovo/BovenmaatFitness.git`
2. `yarn install`
3. `yarn build`
